# 🏦 Jaybhavani Sahakari Bank 

**Complete banking web app** — English / हिंदी / मराठी, images, fully responsive.

## 🚀 Quick Start
```bash
npm install
npm run dev
# Open: http://localhost:5173
```

## 🔐 Login
Go to `/login` — pick Customer or Admin tab.

**Customer:** ramesh@email.com / customer123

## 🌐 Languages
Click the language button (top-right or in sidebar) to switch between English / हिंदी / मराठी.

## 📁 Structure
- `/` — Public website (Home, Accounts, Deposits, Loans, Insurance, Investments, Calculators, Branches, Help, About, Offers)
- `/my` — Customer Portal (Dashboard, Account, Transfer, Statements, Deposits, Loans, Profile)
- `/admin` — Admin Panel (Dashboard, Customers, Loans, Deposits, Transactions, Branches, Staff, Reports, Complaints, Settings)
